# Welcome to Docit

This is a static documentation site generated by **Docit**.

## Features

- Markdown support
- Clean, minimal design
- No JavaScript required
- Responsive layout

## Getting Started

1. Add your markdown files to the docs folder
2. Edit docit.yaml to configure your site
3. Run docit to generate the static site

## Code Example

    func main() {
        fmt.Println("Hello, Docit!")
    }

Enjoy your documentation!
